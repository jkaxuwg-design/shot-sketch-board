# Universal Production Rules

These rules apply to every IP, director, editor, and case unless the current user instruction or locked spreadsheet template says otherwise.

## Script Table

- Use the fixed storyboard columns from `references/script-table-schema.md`.
- Preserve the user's final copy unless asked to rewrite.
- Split by sentence or clear speaking beat.
- Use only allowed dropdown values.
- Write `画面描述` for execution: subject, position, source material, motion, highlight, and transition.

## Material Choice

- Use `录屏素材` for real websites, product pages, desktop software operations, and provided screen assets.
- Use `数字/真人口播` when no external material is needed or the presenter should carry the line.
- Use `动效制作素材` for diagrams, masks, arrows, comparison panels, timelines, icons, and editor-built visual systems.
- Use `AI生成素材` only when the shot needs generated illustration, metaphor, or video material.

## Sketches

- Horizontal 16:9 only.
- Do not sketch `录屏素材`.
- Use blank draft-paper backgrounds.
- Show only required elements.
- For multi-state shots, draw separate states in one row: `前 / 后` or `转场前 / 变化中 / 转场后`.
- Avoid readable text, fake UI, fake code, fake logos, and random glyphs.

## Prompts

- Fill prompt columns for `AI生成素材`.
- Fill prompts for `动效制作素材` only when an AI base frame is needed.
- Leave prompt columns blank for `录屏素材`.
- Chinese and English prompts must describe the same image and motion.

## Review

Run the three-agent review:

- 剪辑 Agent: execution clarity.
- 编导 Agent: visual idea and line relevance.
- 审核 Agent: aspect ratio, hallucination risk, IP contamination, and table value compliance.
